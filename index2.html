웹 화면 구성

레이아웃의 기본 구조는 위와 같습니다.크게 header와 section 2개로 구성되어 있으며 header은 타이틀을 담고 있습니다. section1의 경우 메뉴를 담고 있습니다. section2는 section1에서 메뉴 선택시 높이가 설정되고 상세 페이지를 보여줍니다. section2가 화면에 보이게 되면 section1은 네비의 기능을 가지게 됩니다.

메뉴 화면 구성

menu1의 화면 구조는 왼쪽과 같습니다. 하나의 화면에 내용을 모두 담고 있으며, 타이틀로 나누어 구분됩니다. 메뉴 선택시 그에 맞는 타이틀로 이동합니다.

menu2의 졸업 후 메뉴의 화면 구조는 왼쪽과 같습니다. 두개의 버튼을 담는 블럭과 세개의 블럭을 만들어주었고 화면에 보이지않는 화면은 가로 넓이가 정해져있지 않고 선택됐을 때에 넓이를 가지게 됩니다. 네비의 메뉴를 선택하지 않아도 한 번에 볼 수 있도록 화살표 버튼을 넣어 사용에 편하도록 구성하였습니다.

menu3은 말 그대로 연습장의 역할로 만들어 본 여러 페이지를 볼 수 있도록 하는 메뉴 입니다. 왼쪽에 메뉴바를 두어 오른쪽 화면에서 결과를 볼 수 있습니다.
