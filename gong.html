공모전 이모티콘 애견미용

공모전 도전 계기 컴퓨터 시간에 일러스트레이터와 포토샵을 활용하여 명함 만들기를 한 적이 있었다. 그때 흥미로웠던 기억이 떠올라 도전하게 되었다.

도전 과정공모전 사이트에 올라온 콘테스트 목록 확인 -> 개최자의 니즈에 맞게 디자인을 생각 -> 손으로 스케치 -> 일러스트레이트로 선따기 -> 수정 -> 포토샵 목업 -> 로고 설명으로 올릴 사진 편집 -> 사이트에 올리기
이런 순서로 진행하였는데 생각한 디자인이 나오지않아 속상함을 느꼈었다.

느낀점 흥미롭지만 어려웠던 기억이 컸는데 참가하고자 하는 목표가 생기면서 어렵다는 기억보다 해보고 싶다는 생각이 커졌다. 상상 속 디자인을 구현하지 못한 것에 아쉬움이 남았다. 그래도 어설프게 다룰 줄 알았던 포토샵과 일러스트레이터의 새로운 기능을 알게되었고, 인터넷으로 찾아 배운 것 보다 더 정확하게 이해하고자 책을 구매하였다. 수상은 못했지만 프로그램의 정확한 이해와 사용이 가능하게 되었다.
